1) Generate API key
a unique API Key which will need to be passed to the API in order to use it.
How to generate?
Only admin can do by going to Settings and clicking ‘Enable Access’
*Note:The API Key is unique to the organisation for which it has been created and should be stored in a secure way to prevent unauthorised access to the database

To access the API a tocken needs to generated to activate a session. The acquired tocken then needs to passed in the header of all the subsequent requests to access the APIs.
The tocken has a lifetime and will expire.
